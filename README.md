# The DrawFAVA Project

Welcome to the GitHub page for the DrawFAVA project. For the project website,
please visit [`https://nbenlab.github.io/drawFAVA/`](
https://nbenlab.github.io/drawFAVA/).

The `main` branch of this repository can be used to run the annotation tool.
For more information on this, see the [How to Use the Annotation Tool](
https://nbenlab.github.io/drawFAVA/3_howto/1_setup.html) section of the
[DrawFAVA website](https://nbenlab.github.io/drawFAVA/).

The `website` branch of this repository contains the DrawFAVA website itself.

The `data` branch of this repository contains the DrawFAVA annotations.
